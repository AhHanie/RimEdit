# RimEdit

A free and open source Windows desktop editor for RimWorld mod XML.
Built to make editing Defs and patches easier.

[Releases](https://github.com/AhHanie/RimEdit/releases) | [Issues](https://github.com/AhHanie/RimEdit/issues) | [Discord](https://discord.gg/FvHTybz3q)

## Installation

To install RimEdit, visit the [Releases page](https://github.com/AhHanie/RimEdit/releases) and download the latest zipped Windows release.

Unzip the download and run the RimEdit executable inside the unzipped folder.

RimEdit currently only supports Windows.

## FAQ and Issues

If you run into a bug or have a feature suggestion, create an issue on the [RimEdit Issues page](https://github.com/AhHanie/RimEdit/issues).

For help, discussion, and updates, join the [RimEdit Discord](https://discord.gg/FvHTybz3q).
